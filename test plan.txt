I. Тестирование пустого стека:

1) Создать стек;
2) Попытаться взять из него элемент. Получить None;
3) Посмотреть на size этого стека. Он должен быть равен 0.

II. Тестирование операций push, check:

1) Создать стек;
2) Положить в него n элементов;
3) С помощью check вывести на экран эти элементы. Получить их список.

III. Тестирование операций push, size:

1) Создать стек;
2) Положить в него n элементов;
3) С помощью size вывести на экран размер стека. Он должен быть равен n.

IV. Тестирование операций push, pop:

1) Создать стек;
2) Положить в него n элементов;
3) Взять из него 1 элемент. Содержимое элемента должно появиться на экране;
4) Узнать size этого стека. Он должен быть равен n-1.

V. Тестирование операции clear:

1) Создать стек;
2) Положить в него n элементов;
3) Очистить стек;
4) Попытаться взять из него элемент. Получить None;
5) Посмотреть на size этого стека. Он должен быть равен 0.

VI. Тестирование операции top:

1) Создать стек;
2) Положить в него элемент "a";
3) Вывести значение верхнего элемента. Оно должно быть равно "a";
4) Положить в него элемент "b";
5) Вывести значение верхнего элемента. Оно должно быть равно "b".
